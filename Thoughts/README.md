# 随笔

这里收录了我随性写的文章。从2016年开始，断断续续近三年，篇数不多，但一定程度称得上记录了我这几年思想变化。毕业至今，我做过四份工作，每份工作对于技能要求和之前工作对比都有跨度转折。虽说，这听起来让人觉得我这个人不稳定，会让人起疑：是否浪费了很多时间而没能积淀。然而实际上，我在每份工作中都得到了成长，而且由于工作技能不重复，我的技术视野更开阔了。同时，自己也挺幸运，无意中历经的四个公司的发展也是很不相同，这让我自然而然看到了不同的公司管理、理念对企业前景的影响，多次刷新了我对人才的认识。

上次写文章应该是一年前了吧，今年倒是积攒了零零散散的题材，但都懒得整理，也就没有产出了。现在，既然正式的给自己弄了个随想笔记本，就应该端端正在做点什么。首先呢，我会挪来之前写的非技术文章，以前写过些技术文章，但我相信自己能写出更好的，便将它们都丢弃了。因此，挪过来文章的主要是之前的一些思考，当然，那些思考是建立在那之前经验的基础上。然后，我将整理今年的经验，就像重构代码，当然，整理这些经验需要时间。今年是我收获比较大的一年，我对此很期待。
